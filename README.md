# User Panel

User Panel for Priston Tale servers.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install this project.

```bash
npm install
```

## Usage

```zsh
npm run dev

```

## Build

```zsh
npm run build

```

## Tests

```bash
npm run test

```
